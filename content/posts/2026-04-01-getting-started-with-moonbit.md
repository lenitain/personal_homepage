MoonBit is a modern programming language designed for WebAssembly, embedded systems, and cloud-native applications.

## Why MoonBit?

MoonBit combines the safety of Rust with the simplicity of Python. Its type system catches errors at compile time, while its syntax remains approachable for newcomers.

```moonbit
fn greet(name : String) -> String {
  "Hello, \{name}!"
}
```

## Key Features

- **Fast compilation**: MoonBit compiles to WebAssembly with near-instant build times
- **Pattern matching**: Elegant handling of complex data structures
- **Zero-cost abstractions**: High-level code without runtime overhead

## Getting Started

Install MoonBit via the official installer, then create your first project:

```bash
moon new my_project
cd my_project
moon run
```

MoonBit is still evolving rapidly. The community is growing, and the ecosystem is expanding with new packages every week.
